# Personal Website

A minimalist personal website starter built with Jekyll, focused on performance and SEO.

## Features

- **Fast Loading**: Static HTML with minimal CSS, no unnecessary JavaScript
- **SEO Optimized**: Jekyll SEO Tag, Open Graph, sitemap, robots.txt
- **Accessible**: Semantic HTML, ARIA labels, keyboard navigation
- **Mobile-Friendly**: Responsive design that works on all devices
- **Easy to Maintain**: Just write Markdown files

## Technology Stack

- Jekyll 4.3
- Pure CSS (no frameworks)
- GitHub Pages ready

## Key Highlights

- Top navigation with About, Talks, Articles, and Projects
- Collections for organizing content
- Automatic sitemap generation
- RSS feed support
- Social media preview images
